使用hexo+gihub搭建博客记录

  • hexo

下午搞了下hexo和github，搭建了个人的博客站点。博客的建立也是激励自己，在2016年里好好努力，进行更多的技术积累和个人沉淀。

hexo确实好用，简单方便，网上的资料也很多。不过在搭建的过程中也遇到了一些问题，在此记录一下。

后记：在2015-12-29日搭建了这个博客。今天是2016-5-26日。2016年已快过了一半了，原来在2015年末想要做的事情貌似都没有做。我仍然是处于定计划却不实施的阶段哈。今天想使用github做在线demo，想起来这个博客，可是却不知道如何使用了。所以把这篇文章完善下吧。

环境：

  • nodejs
  • git
  • hexo
