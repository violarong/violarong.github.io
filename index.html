使用hexo+gihub搭建博客记录

下午搞了下hexo和github，搭建了个人的博客站点。博客的建立也是激励自己，在2016年里好好努力，进行更多的技术积累和个人沉淀。

hexo确实好用，简单方便，网上的资料也很多。不过在搭建的过程中也遇到了一些问题，在此记录一下。

环境：

  • nodejs
  • git
  • hexo

nodejs和git，环境中就已经装好，不需要折腾。按照hexo官网，安装了下hexo。到此一切顺利。hexo server可以正常启动本地服务。现在需要将它与github进行关联，使用http://username.github.io 就可以访问站点。没有自己买域名，就直接使用github.io了。一切按教程，到github建仓库。然后，hexo generator,hexo deploy。发现可以generator成功，deploy没有反应。网上找了issue，原来是自己没有装git插件。npm install hexo-deployer-git –save就可以deploy了。注意：需要增加–save，否则还是无法deploy。打开站点http://violarong.github.io/，可以正常访问到了。
